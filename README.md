
# HTML, CSS e JavaScript — Diário de Estudos

Este repositório documenta o progresso dos estudos em HTML, CSS e JavaScript. O objetivo é reunir exemplos, exercícios e pequenas aplicações que ilustram conceitos aprendidos ao longo do estudo, servindo como referência prática e registro de evolução.

## Como usar
- Abra os arquivos HTML diretamente no seu navegador para visualizar os exemplos.
- Arquivos .css e .js relacionados estão na mesma pasta e são carregados pelas páginas HTML correspondentes.
- Exemplos simples permitem modificar o código e testar rapidamente mudanças.

## Estrutura do repositório
- [buttons.css](buttons.css) — estilos para exemplos de botões.
- [buttons.html](buttons.html) — exemplos visuais de botões.
- [calculadora.html](calculadora.html) — interface HTML da calculadora de exemplo.
- [calculadora.js](calculadora.js) — lógica JavaScript da calculadora.
- [div.css](div.css) — estilos para blocos de conteúdo (ex.: classe [`.feedback-box`](div.css)).
- [div.html](div.html) — exemplo de uso dos estilos em [div.css](div.css).
- [html-basic.html](html-basic.html) — exemplos básicos de marcação HTML.
- [index.js](index.js) — scripts de demonstração/apoio.
- [lists.html](lists.html) — exemplos de listas em HTML.
- [README.md](README.md) — este arquivo.
- [sign-up-form.html](sign-up-form.html) — formulário de inscrição de exemplo.
- [sign-up.css](sign-up.css) — estilos do formulário de inscrição.
- [teste.html](teste.html) — arquivo de testes/experimentos.
- [text.html](text.html) — exemplos de formatação de texto.

## Boas práticas
- Comentar trechos de código ao experimentar.
- Manter versões de arquivos ao fazer mudanças significativas.
- Usar o repositório como um espaço de aprendizagem: commits pequenos e descritivos ajudam a acompanhar o progresso.
